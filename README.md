# SIOP Reporting Dashboard

## Dashboard Preview

![Solution Report](report.png.png)

Interactive Power BI dashboard for Sales, Inventory, and Operations Planning (SIOP) analysis using DAX, Power Query, and data visualization.

## Tools Used
- Power BI
- Power Query
- DAX
- Excel

## Features
- Forecast Accuracy & Forecast Bias analysis
- Revenue, Non-Revenue and CAPEX dashboards
- Snapshot comparison (Resultant, Lag 0, Lag 1)
- Predictive forecasting using Exponential Smoothing
- Interactive slicers and KPI cards
